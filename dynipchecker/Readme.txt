About
DynIPchecker project is for obtaining your computer dynamic IP   

Instalation
1 - upload files to server
2 - create db und upload schema.sql
3 - enter your server name and unique token to sendip.sh script
4 - enter db credentials and same token in ip/config.php file
5 - add sendip.sh script to crontab (0 * * * * * path_to_script/sendip.sh)
6 - check your ip at http://your_website/ip/
